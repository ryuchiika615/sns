リュッター (Ryutter)

エンジニア就活用のポートフォリオとして開発した学習SNSです。

🚀 アプリURL
[https://ryunosuke-study-sns.onrender.com/](https://ryunosuke-study-sns.onrender.com/)

🛠️ 主な機能
- 日々の学習記録: 科目別に勉強時間をグラフ化・可視化。
- **学習分析レポート: 科目ごとの割合や月ごとの推移をチャートで表示。
- ゲーミフィケーション: ガチャを回して称号やアイコンを獲得し、プロフィールをカスタマイズ。
- SNS機能: 投稿へのいいね、コメント、フォロー・フォロワー機能。

💻 使用技術
- Backend: Python, Django
- Frontend: HTML, CSS, JavaScript, Chart.js
- Database: PostgreSQL
- Deployment: Render
